🎉 Event Planner - Flutter

Event Planner is a cross-platform mobile application built with Flutter. It allows users to book venues, restaurants, and event services for weddings, birthdays, and other celebrations. The app provides a seamless experience with a modern UI, advanced search, and Firebase integration.

👉 Features

Venue Booking: Find and reserve event halls based on location, price, and rating.

Restaurant Reservations: Book seafood, meat, and other restaurant types.

Additional Services: Hire photographers, makeup artists, and order cakes for events.

Search & Filtering: Sort venues by price, location, and capacity.

User Reviews & Ratings: View and submit feedback.

Push Notifications: Get updates on bookings and offers.

🔧 Tech Stack

Flutter + Dart for UI and cross-platform support

Firebase Firestore for data storage

Google Maps API for location services

Dio for API integration

Riverpod / Bloc for state management
